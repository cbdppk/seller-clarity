# Skill: Hackathon Next.js Shipping

## Goal
Ship a reliable Next.js + TypeScript MVP quickly.

## Use this skill when
- bootstrapping the project
- creating pages fast
- wiring UI to API
- preparing for deployment

## Workflow
1. Define the shared contract first
2. Build the home screen
3. Build the results shell
4. Add a fake/mock analyzer
5. Connect the API route
6. Add loading/error states
7. Deploy early

## Rules
- keep one clean user flow
- avoid auth/database for the hackathon
- make the happy path work first
- optimize for phone screens
