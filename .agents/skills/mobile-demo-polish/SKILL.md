# Skill: Mobile Demo Polish

## Goal
Make the MVP feel polished on a phone-sized screen.

## Use this skill when
- polishing spacing
- improving readability
- simplifying charts
- preparing demo flow

## Workflow
1. Check the home screen on narrow width
2. Make the main CTA obvious
3. Keep cards readable
4. Reduce dense text
5. Ensure the sample path always works

## Rules
- prioritize clarity over decoration
- one screen should tell one story
- charts must remain readable on small widths
